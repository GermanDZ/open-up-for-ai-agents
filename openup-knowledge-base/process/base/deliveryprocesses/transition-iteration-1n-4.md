Description

In the Transition phase, the main objectives are to fine-tune the functionality, performance, and overall quality of the beta product from the end of the Construction phase.  The following table summarizes the Transition phase objectives and what activities address each objective:  **Transition phase objectives and activities**

|  **Phase objectives** |  **Activities that address objectives**
---|---
Beta test to validate that user expectations are met  |  [Ongoing Tasks](../capabilitypatterns/ongoing-tasks-19.md) [Develop Solution Increment](../capabilitypatterns/develop-solution-increment-2.md) [Test Solution](../capabilitypatterns/test-solution-2.md)
Achieve stakeholder concurrence that deployment is complete  |  [Plan and Manage Iteration](../capabilitypatterns/plan-and-manage-iteration-12.md) [Test Solution](../capabilitypatterns/test-solution-2.md)
Improve future project performance through lessons learned  |  [Plan and Manage Iteration](../capabilitypatterns/plan-and-manage-iteration-12.md)
