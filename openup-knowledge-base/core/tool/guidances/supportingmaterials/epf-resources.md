Main Description

To learn more about the Eclipse Process Framework \(EPF\), read Per Kroll's article, [The Eclipse Process Framework project](http://www.ibm.com/developerworks/rational/library/05/1011_kroll/index.html) \(IBM® developerWorks®, November 2005\)  For more information on EPF Composer, see the following:
  * [IBM Rational Method Composer: Part 1: Key concepts](http://www.ibm.com/developerworks/rational/library/dec05/haumer/) by Peter Haumer \(IBM developerWorks, December 2005\)
  * EPF Composer Overview, [Part 1](http://www.eclipse.org/epf/general/EPFComposerOverviewPart1.pdf) and [Part 2](http://www.eclipse.org/epf/general/EPFComposerOverviewPart2.pdf) . \([Get Adobe Reader](http://get.adobe.com/reader/)\)
  * For an in-depth explanation of the metamodel on which EPF Composer is based, see OMG: [Software Process Engineering Meta-Model,](http://www.omg.org/technology/documents/formal/spem.htm) Version 2.0, 2008.
