Main Description

The purpose of this discipline is to evolve a robust architecture for the system.  The Architecture discipline is related to these other disciplines:
  * The [Requirements](requirements-2.md) discipline provides the architecturally significant requirements.
  * The [Development](development-1.md) discipline designs and implements the architecture.
  * The [Test](test-1.md) discipline verifies the stability and correctness of the architecture.
  * The [Project Management](project-management-1.md) discipline plans the project and each iteration.
