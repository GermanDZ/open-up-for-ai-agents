Check Items

Has the plan been elaborated with the team? |  Project planning should not be done in isolation, and every member of the team should be involved.
---

Has the team agreed on development practices for the project?

Who plays which roles?  How long will the iterations be?  What coding standards should be followed?  Has the project directory structure been identified?  What development, build, and test tools should be used?  What measures and metrics will be used to track progress?
---

Have the project milestones been defined?

What are the major milestones for each release?
---

Has the deployment strategy been discussed?

How will the product be delivered to the customer?
