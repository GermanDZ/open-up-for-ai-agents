Main Description

###  Step-­by-­step outline

  1. Insert card
  2. Enter PIN
  3. Select transfer
  4. Select amount
  5. Select from account
  6. Select to account
  7. Send transaction
  8. Receive confirmation
  9. Print receipt
  10. Eject card

###  List of Alternative flows

A1 Wrong PIN
A2 No money
A3 From account does not exists
A4 To account does not exists
A5 No confirmation
A6 Card stuck
A7 Out of paper
A8 Card stuck
