 This work product represents all the materials needed to train end users and production support personnel on the features and inner workings of a product for a particular release.
---

Purpose

The purpose of this work product is to provide direct understanding of how to use or support a set of features deployed in a particular release.

Main Description

Training materials that can be used to train end users and production support personnel might be of the following types:
  * Presentation slides
  * Handouts
  * Job aids
  * Tutorials
  * On-line demos
  * Video vignettes
  * Lab exercises
  * Quizzes
  * Workshop materials, etc.
